# このソフトウェアについて #

HelloDrawString20160630は私個人が学習目的で作成したソフトウェアです。

C++/win32のGDI+で文字列を描画しました。

# 開発環境 #

* Windows XP SP3
* C++/Win32
* Visual C++ 2010 Express
